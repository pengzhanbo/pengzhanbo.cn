类型推导

约 140 字小于 1 分钟

2022-06-03

Rust 是一门 静态类型语言。

编译器必须在编译器知道我们所有变量的类型，但这并不意味着需要为每个变量指定类型。

因为 Rust 编译器很聪明，它可以根据变量的值和上下文中的使用方式来自动推导出变量的类型 ，同时编译器也不够聪明，在某些情况下，它无法推导出变量类型，需要手动去给予一个类型标注。
