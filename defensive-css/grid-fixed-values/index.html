Grid fixed values

假设我们有一个 网格 布局，包含一个 aside 和 一个 main。 CSS 如下所示：

.wrapper {
  display: grid;
  grid-template-columns: 250px 1fr;
  gap: 1rem;
}
aside
main

由于空间不足，这将在较小的视口尺寸上导致溢出。 为避免此类问题，请在使用上述 CSS 网格时始终使用媒体查询。

在视口尺寸较小时，换行显示，在视口尺寸较大时，使用 网格布局。

@media (min-width: 600px) {
  .wrapper {
    display: grid;
    grid-template-columns: 250px 1fr;
    gap: 1rem;
  }
}
aside
main

调整浏览器窗口大小查看效果。
